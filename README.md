# M3Uzhuanhuaqi
用于将获取配置失败的txt直播源文件转化为能够直播的工具

🚀 功能说明
- 自动扫描文件夹中的所有txt文件
- 提取频道信息（标题+URL）
- 转换为标准M3U格式
- 每200个频道分割为一个文件

📁 文件结构
M3U转换器.py       - M3U格式转换器（推荐使用）

💡 使用步骤
1. 将需要转换的txt文件放入"转化文件"文件夹
2. 双击运行"M3U转换器.py"
3. 等待处理完成
4. 在文件夹中查找生成的m3u文件
5. 将生成的M3U文件导入ok影视pro

📊 输出格式（M3U版本）
- 文件名：自行修改
- 格式：标准M3U格式，#EXTM3U + #EXTINF + URL
- 大小：每文件约25-35KB
- 容量：每文件200个频道

🔧 系统要求
- Windows系统
- 已安装Python 3.12（系统已配置）

📞 技术支持
- M3U格式兼容性更好，推荐优先使用
